# CRUD Product Management System

A simple product management system built using HTML, CSS, and JavaScript.

## Featuresproducts
- Delete products
- Delete all products
- Search by title
- Search by category
- Local Storage support
- Total price calculation

## Technologies Used

- HTML
- CSS

- Add products
- Display products
- Update 
- JavaScript
- Local Storage
